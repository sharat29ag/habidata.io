Pipeline

Four steps.
Zero friction.

01

Select Project

Point HabiDATA at your camera-trap folder. It scans for images, videos, and any existing results.

02

Run Detection

Choose model weights, set confidence threshold, and hit Run. A progress bar tracks inference across all files.

03

Edit & Tag

Walk through detections image-by-image. Correct labels, draw boxes, assign species, and mark as verified.

Offline-First

All inference runs locally. Your images never leave your machine â€” critical for field stations with limited connectivity and data-sharing restrictions.

Append-Only Ledger

Edits are appended as NDJSON at the byte level â€” O(1) saves regardless of dataset size. A 3-second debounce flush consolidates changes.

Crash-Safe Writes

Detection results go through SQLite WAL-mode for atomic batch writes, even on external drives. JSON is rebuilt after inference completes.
