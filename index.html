Pipeline

A streamlined four-stage workflow.
From folder to finished analysis.

01

Select Project

Point HabiDATA at your camera-trap folder. The application scans for images, videos, and any existing results from prior sessions.

02

Run Detection

Select model weights, configure the confidence threshold, and begin inference. A progress bar tracks processing across all files in the project.

03

Edit & Tag

Review detections image-by-image. Correct labels, resize bounding boxes, assign species from taxonomic lists, and mark images as verified.

04

Analyse & Export

Examine capture patterns through interactive charts. Generate deployment sheets and export summary visualisations for reports and manuscripts.

Offline-First

All inference runs locally on your hardware. Images never leave your machine â€” essential for field stations with limited connectivity and projects with data-sharing restrictions.

Crash-Safe Persistence

Detection results are written through SQLite in WAL mode, ensuring atomic batch writes even on external drives. The canonical JSON is rebuilt only after inference completes successfully.

Cross-Platform

Runs on Windows, macOS, and Linux. Pre-built binaries are available for Windows. macOS users benefit from MPS-accelerated inference on Apple Silicon.
